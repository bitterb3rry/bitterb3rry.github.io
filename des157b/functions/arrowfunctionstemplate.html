Ways to write functions

This function checks the time on videos from my mini studio 1 to time captions that show up.

Named Function

            function checkTime() {
                if (index == 0) {
                    for (let i = 0; i < ketchup.start.length; i++) {
                        if (ketchup.start[i] < myVideo.currentTime && myVideo.currentTime < ketchup.stop[i]) {
                            caption.innerHTML = ketchup.line[i];
                        }
                    }
                }
            }
